The <section> element

Parent Specification: HTML 5.1 (Working Draft)

The section element represents a generic section of a document or application. A section, in this context, is a thematic grouping of content. Each section should be identified, typically by including a heading (h1-h6 element) as a child of the section element.

Sample Code:

<section>
    <h1>Section Title</h1>
    <p>Text that describes this section.</p>
</section>

Modern versions of assistive technologies will announce CSS generated content, as well as specific Unicode characters. To avoid unintended and confusing output in screen readers (particularly when icons are used purely for decoration), we hide them with the aria-hidden="true" attribute.

Type-Retention

This test determines whether the "number" type is retained on an input element. If it is not, the input type is unsupported. In some cases, type-rentention could result in false-positives.

function test () {
	var el = document.createElement( "input" );
	el.type = "number";
	return "number" === el.type;
}
